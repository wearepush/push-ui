## About

The starter ui includes the following technologies:

| Technology                                    | Description                                                                    |
| --------------------------------------------- | ------------------------------------------------------------------------------ |
| [Node 16.0.0 LTS](https://nodejs.org/en/)     | Node                                                                           |
